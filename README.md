# cool-conky

A cool conky theme.

### Features

- Uptime
- Load
- CPU (usage/cores load)
- Memory (usage)
- GPU (temperature, frequency)
- Disk (usage, read/write)
- Ethernet (local IP, up/down speed)
- Processes (top 5, usage)
